# Modul: Encode

## str_hex(string)
arg: string

return: hexcode

Example:
```python
from decode_things import encode

hex = encode.str_hex("abc")
print(hex)
```

## int_hex(integer)
arg: integer

return: hexcode

Example:
```python
from decode_things import encode

hex = encode.int_hex(123)
print(hex)
```
